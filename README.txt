== Description ==

Curtain Raiser for Inaugural Ceremony is a simple, yet effective plugin that adds a curtain-raiser feature to your WordPress website. 

In India, inaugurating an official website with a ceremony is a common practice for NGOs and Government bodies. This plugin caters to the need for adding a curtain-raiser feature for such inaugural ceremonies. 

To activate, simply add ?curtain_ceremony=true to your URL after installing the plugin, and witness the curtain-raiser functionality in action.

To test the functionality, [click here](https://tastewp.com/new?pre-installed-plugin-slug=wp-curtain-raiser&redirect=plugins.php&ni=true) to spin up a new Tastewp Instance.

== Installation ==

Follow these simple steps to install the plugin:

1. Upload the contents of 'wp-curtain-raiser' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Once activated, add ?curtain_ceremony=true to your URL to view the curtain-raiser functionality.

== Frequently Asked Questions ==

= How do I use the curtain-raiser feature on the homepage? =

After activating the plugin, add ?curtain_ceremony=true to the homepage URL in the address bar.

= I've activated the plugin but can't see the curtains or the rope. What should I do? =

This issue might occur due to a CSS conflict with other plugins. As a quick fix, try adjusting the z-index values of the conflicting elements. If you're unable to resolve it, please post your URL in the WordPress Support forum for assistance.

== Upgrade Notice ==
If you encounter any issues while upgrading, please delete the plugin from the Plugins folder and reinstall a fresh copy. We do not store any settings in the database. This plugin is intended for single-day use, and we recommend deactivating it after the ceremony. Keeping unnecessary plugins active is not a best practice.
